CSS | How I
Style My Web Pages

CSS transforms plain HTML into polished, responsive interfaces. I have built solid skills in the Box Model, Flexbox, CSS Grid, custom properties (variables), media queries for responsiveness and keyframe animations, all of which I use in real projects.

The Box Model

Every element I build is a box. Understanding margin, border, padding and content is the foundation of all CSS layout work.

/* I always use border-box so padding does not unexpectedly break widths */ * { box-sizing: border-box; } .my-card { width: 300px; padding: 24px; /* inside the border */ border: 2px solid #ccc; margin: 16px; /* outside the element */ }

Flexbox | My Go-To Layout Tool

I use Flexbox constantly for navbars, card rows and centering content. It replaced all the float hacks I used to rely on.

/* My card row pattern */ .card-row { display: flex; gap: 20px; flex-wrap: wrap; justify-content: space-between; align-items: stretch; } .card { flex: 1; min-width: 200px; } .centred { display: flex; justify-content: center; align-items: center; }

CSS Variables

I define all colours and spacing as variables so the entire theme changes from one place.

/* Top of every stylesheet */ :root { --primary: #c88860; --bg: #faf5ee; --radius: 14px; --shadow: 0 4px 24px rgba(0,0,0,.1); } .button { background: var(--primary); border-radius: var(--radius); } @media (prefers-color-scheme:dark) { :root { --bg: #1e1810; } }

Animations | Pure CSS, Zero JavaScript

/* Hover transition on my cards */ .card { transition: transform .28s, box-shadow .28s; } .card:hover { transform: translateY(-4px); box-shadow: 0 14px 44px rgba(0,0,0,.14); } /* Float animation on my profile photo */ @keyframes floatY { 0%,100% { transform: translateY(0); } 50% { transform: translateY(-14px); } } .profile { animation: floatY 5s ease-in-out infinite; }
